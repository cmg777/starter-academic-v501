# Website Logs

This directory documents the current status and evolution of carlos-mendez.org.

Use it to track significant changes, ongoing work, and the overall trajectory of the site. When making notable updates (new sections, structural changes, performance improvements, theme upgrades), add or update entries here to maintain a clear historical record.

## Suggested format

Each log entry can be a markdown file named by date or topic, e.g.:

- `2026-02-10-performance-optimizations.md`
- `2026-02-10-dashboard-page-setup.md`

Include what was changed, why, and any follow-up items.
