## What was created

Stata companion to the R and Python FWL tutorials, using the scatterfit package (Ahrens, 2024) for FWL-residualized scatter plots. Uses the same three datasets (loaded from GitHub URLs) for cross-language comparability. Third in the FWL trilogy.

**Key results (match R and Python tutorials):**
- Naive coupon coefficient: -0.093 (wrong sign), controlled: +0.212 (true: +0.2)
- OVB formula: 0.300 x -0.494 = -0.148
- FWL manual verification: 0.212288 (exact match to 6 decimal places)
- Flights: air time coefficient -0.005 (no FE) to -0.032 (origin + dest FE)
- Wages: R-squared 0.148 (pooled) to 0.617 (individual FE)

**Stata-specific features showcased:**
- Binned scatter plots (`binned` option)
- Regression parameters on plots (`regparameters(coef se pval r2 n)`)
- `controls()` for continuous FWL, `fcontrols()` for categorical FE
- Cross-language comparison table (Python vs R vs Stata syntax)
