# ESDA Tutorial Post: Spatial Clusters and Dynamics of Human Development in South America

**Date:** 2026-03-22

## Summary

Added an Exploratory Spatial Data Analysis tutorial (`content/post/python_esda2/`) using PySAL libraries. Analyzes spatial clusters of subnational human development across 153 South American regions (2013 and 2019) from the Global Data Lab. Covers scatter plots, choropleth maps (Fisher-Jenks classification), spatial weights, global and local Moran's I, LISA cluster maps, and space-time dynamics via directional Moran scatter plots. Companion to the Pooled PCA post (`python_pca2`).

## Key files

- `content/post/python_esda2/index.md` -- full tutorial (14 sections, ~600 lines)
- `content/post/python_esda2/script.py` -- standalone Python script
- `content/post/python_esda2/data.geojson` -- sub-national boundaries with SHDI indicators
- `content/post/python_esda2/esda2_*.png` -- 9 dark-theme figures
- `content/post/python_esda2/featured.png` -- LISA 2019 cluster map

## Key results

- Moran's I increased from 0.568 (2013) to 0.632 (2019) -- spatial clustering strengthened
- LISA identifies 30 HH clusters (Southern Cone) and 37 LL clusters (Amazon/north) in 2019
- LL cluster expanded from 29 to 37 regions (Venezuelan crisis spillovers)
- 87% of HH regions persisted; 62% of LL regions persisted
- 2 island regions detected (San Andres, Nueva Esparta)

## Technical notes

- Used manual LISA scatter plots (replacing splot's `moran_scatterplot`) for correct quadrant colors matching the cluster map (HH=red, LL=blue, HL=orange, LH=lightblue)
- Used manual global Moran scatter plots for visible quadrant lines on dark theme
- Fisher-Jenks breaks from 2013 held constant for 2019 maps with region counts in legend
- `adjustText` library for labeling extreme regions on scatter plots and maps
- Created `region_country` column merging simplified region names with ISO country codes
- `giddy` Rose diagram replaced with manual directional Moran scatter plot (numba/llvmlite build issue)
- Islands excluded from scatter plots (spatial lag = 0 is not meaningful for regions with no neighbors)
