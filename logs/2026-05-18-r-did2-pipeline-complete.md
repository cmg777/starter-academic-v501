# r_did2 — Full DiD pipeline complete

**Date:** 2026-05-18

**Topic:** Difference-in-Differences for regional data (Medicaid expansion + adult mortality)

## Summary

The `r_did2` post is the first full end-to-end pass through the eight-skill
data-science pipeline for an R-based causal-inference tutorial. The empirical
case study replicates Baker, Callaway, Cunningham, Goodman-Bacon & Sant'Anna's
(2025) flagship Medicaid-expansion example and threads "weighting changes the
*target parameter*" through every stage. All five deliverable artefacts now
live in the page bundle; the post is publishable and renders locally as a
self-contained Quarto notebook.

## Highlights

- **Sign-reversal story:** unweighted 2x2 ATT(2014) = +0.122 deaths per
  100,000; population-weighted = -2.563. The reversal survives DRDID,
  staggered 2xT/GxT cohorts, and is bracketed by HonestDiD relative-magnitudes
  bounds (unweighted breaks at M̄ ≈ 0.25; weighted bound straddles zero from
  M̄ = 0 already).
- **22 R code chunks, 5 display equations, 1 Mermaid roadmap, 12 concept-card
  toggles, 8 dark-theme PNGs.** All numerics traced back to `summary.csv`.
- **Quarto pinning:** `tutorial.qmd` pins 13 packages via
  `pak::pkg_install("pkg@x.y.z")` from the local R 4.5.2 environment (dplyr
  1.1.4, ggplot2 4.0.1, fixest 0.14.0, did 2.3.0, DRDID 1.2.3, HonestDiD
  0.2.8, etc.). First-attempt render produced a 266 KB HTML with zero math
  errors; all 6 headline numbers (0.122 / -2.563 / -1.226 / -3.756 / 7.917 /
  0.266) verified in the rendered output.
- **Three review passes** (`/project:review-script`, `/review-results-report`,
  `/review-post`) all returned ACCEPT verdicts; the only outstanding items
  are the deferred infographic stage and a user-managed `featured.png` (the
  bundle currently ships `featured.webp`).
- **Citation polish:** all manuscript references updated to the full
  bibliographic entry "Baker, A., Callaway, B., Cunningham, S.,
  Goodman-Bacon, A., & Sant'Anna, P. H. (2025). *Difference-in-Differences
  Designs: A Practitioner's Guide.* arXiv preprint arXiv:2503.13323."
