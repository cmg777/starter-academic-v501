## Gotchas
- Don't translate query keys: project `tags` (the `/es/` filter) and author `user_groups`
  (must match `content/es/home/people.md`).
- Tutorial stubs use a custom `card_url` key (NOT the reserved `url:`) + `_build: {render:
  never, list: always}`.
- Hugo window: ≥0.96 (`continue`), < 0.120 (`site.GoogleAnalytics`), < 0.128 (`paginate`).
