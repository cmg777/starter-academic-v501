# PCA Tutorial Post: Building Development Indicators

**Date:** 2026-03-21

## Summary

Added a data science tutorial on Principal Component Analysis for building composite development indicators (`content/post/python_pca/`). The post constructs a Health Index from Life Expectancy and Infant Mortality using simulated data for 50 countries, walking through a 6-step manual PCA pipeline and verifying against scikit-learn. Includes a Colab-ready notebook, standalone script, and a reusable general-purpose PCA pipeline.

## Reusable PCA pipeline (Section 12)

The scikit-learn pipeline is designed as a general-purpose script configurable via 4 variables:

```python
CSV_FILE = "https://..."     # Path or URL to CSV
ID_COL = "country"           # Row identifier column
POSITIVE_COLS = ["life_exp"] # Higher = better
NEGATIVE_COLS = ["infant_mort"] # Higher = worse (will be flipped)
```

Summary output uses `ID_COL`, `indicator_cols`, and generic labels ("Top 5:", "Bottom 5:") so the pipeline works with any dataset without code changes beyond the config block. The CSV is loaded from the GitHub raw URL so the notebook runs standalone in Colab.
