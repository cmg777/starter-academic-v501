# Data dictionary - California's Proposition 99: The Synthetic DiD Panel

Companion data for a hands-on Stata tutorial on <strong>synthetic difference-in-differences (SDID)</strong>, applied to re-evaluate <strong>California's Proposition 99</strong> &mdash; the 1988 ballot measure that raised the cigarette excise tax by 25 cents a pack and funded an anti-smoking campaign. The file is the canonical strongly balanced panel distributed with the <code>sdid</code> package (originally from Abadie, Diamond &amp; Hainmueller 2010, and used by Arkhangelsky et&nbsp;al. 2021): <strong>39 US states</strong> observed annually from <strong>1970&ndash;2000</strong> &mdash; 1,209 observations &mdash; with annual cigarette sales in <strong>packs per capita</strong> as the sole outcome. California is the single treated unit; the policy bites from 1989 onward. The post writes DiD, synthetic control, and SDID as one weighted two-way fixed-effects regression and estimates the ATT of Proposition&nbsp;99 with the <code>sdid</code> command, cross-checking synthetic control against <code>synth2</code>.


Companion data for the post <https://carlos-mendez.org/post/stata_sdid/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Single treated unit.</strong> Only 12 of 1,209 observations are treated (California, 1989–2000). With one treated unit, placebo (permutation) inference is the only valid procedure — the jackknife is undefined and the bootstrap is unreliable; statistical power is inherently limited.
- <strong>Outcome-only panel.</strong> The file carries no income, price, or demographic covariates — only cigarette packs per capita. This is deliberate: synthetic control and SDID see exactly the same information, so any difference in their answers comes from the estimator, not from different predictors.
- <strong>Strongly balanced.</strong> Every state is observed in every year 1970–2000 with no gaps; this balance is required by all three estimators in the post.
- <strong>Observational, not experimental.</strong> California was not randomly assigned Proposition 99; identification assumes a stable comparison group, no large contemporaneous shock unique to California in 1989, and no cross-state spillovers (e.g., border cigarette purchases contaminating donor states).
- <strong>Distributed dataset.</strong> Values are byte-faithful to the canonical prop99_example.dta shipped with the sdid package; this data dictionary adds only metadata (variable + value labels).
