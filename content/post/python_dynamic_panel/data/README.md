# Data dictionary - Dynamic Panel Data Models: Employment Persistence

Companion data for a hands-on Python tutorial that estimates how persistent firm-level employment is &mdash; the autoregressive coefficient <code>&rho;</code> of a dynamic labor-demand equation &mdash; using the canonical <strong>Arellano and Bond (1991)</strong> panel of 140 UK manufacturing firms observed annually over 1976&ndash;1984 (1,031 firm-years, unbalanced). The tutorial walks the full estimator ladder: pooled OLS (biased up by the omitted firm effect) and fixed effects (Nickell bias, biased down) via <code>pyfixest</code>, then Anderson&ndash;Hsiao IV, Arellano&ndash;Bond difference GMM and Blundell&ndash;Bond system GMM via <code>pydynpd</code>, with the AR(2), Hansen, and instrument-collapse diagnostics that separate the one defensible estimate (system GMM, <code>&rho;&#770; = 0.927</code>) from four seductive wrong ones. This dataset is the classic dynamic-panel teaching dataset, used by Arellano &amp; Bond, Blundell &amp; Bond (1998), and Roodman (2009).


Companion data for the post <https://carlos-mendez.org/post/python_dynamic_panel/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Teaching dataset.</strong> This is a methods showcase on 1970s&ndash;80s UK manufacturing, not a current estimate of employment dynamics; the value is in the workflow, not the era-specific numbers.
- <strong>Each lag burns data.</strong> With T as small as 7&ndash;9 per firm, every lag/difference costs each firm its first observed year(s): the panel falls from 1,031 rows to 891 (one lag), to 751 (GMM), to 611 (the two-lag replication spec). The lag/difference columns in <code>data_prepared</code> are missing for those dropped firm-years.
- <strong>Unbalanced panel.</strong> 103 firms appear for 7 years, 23 for 8, and 14 for all 9; estimators handle this natively but observation counts vary by specification.
- <strong>Variance is lopsided.</strong> The between-firm SD of log employment (1.339) is about seven times the within-firm SD (0.195), so the unobserved firm level dominates &mdash; which is exactly why naive estimators fail and why dynamic-panel GMM is needed.
- <strong>Headline caveat.</strong> The system-GMM 95% CI [0.773, 1.081] includes the unit root, so 'employment is stationary' is not a defensible claim &mdash; only the point estimate and its lower bound are.
