3. BASELINE: pooled OLS vs fixed effects - the bias bracket
======================================================================
Model: n ~ L1.n + w + L1.w + k + L1.k + year dummies
Pooled OLS ignores alpha_i: L1.n is positively correlated with the
omitted firm effect, so rho is biased UPWARD.
Fixed effects (within) removes alpha_i but creates a mechanical
negative correlation between demeaned L1.n and the demeaned error -
the Nickell (1981) bias, of order 1/T, biasing rho DOWNWARD.

4. ANDERSON-HSIAO: first-difference IV - right idea, noisy answer
======================================================================
First-differencing kills alpha_i but makes d.L1.n endogenous (it
contains eps_i,t-1). Anderson-Hsiao (1981): instrument d.L1.n with
the LEVEL n_i,t-2, which predicts the difference but - if eps is not
serially correlated - is uncorrelated with d.eps_it.

Consistent in theory, useless in practice here: one instrument, and
the confidence interval swallows the whole OLS-FE bracket. GMM's
insight: EVERY deeper lag is also a valid instrument - use them all.

7. INSTRUMENT PROLIFERATION: lag windows vs collapse
======================================================================
GMM's blessing is its curse: with T=9, 'use every lag' generates
instruments quadratically in T. Too many instruments overfit the
endogenous variables and weaken the Hansen test (p -> 1 is a red
flag, not a pass). Roodman's rule of thumb: instruments < N groups.

  lags 2:3   collapse=False ->  68 instruments, rho = 0.956, Hansen p = 0.035
  lags 2:3   collapse=True  ->  17 instruments, rho = 0.921, Hansen p = 0.096
  lags 2:5   collapse=False ->  95 instruments, rho = 0.935, Hansen p = 0.186
  lags 2:5   collapse=True  ->  23 instruments, rho = 0.937, Hansen p = 0.255
  lags 2:99  collapse=False -> 113 instruments, rho = 0.930, Hansen p = 0.235
  lags 2:99  collapse=True  ->  32 instruments, rho = 0.927, Hansen p = 0.462

Read the uncollapsed column top to bottom: as instruments pile up the
Hansen p-value drifts steadily upward (0.04 -> 0.19 -> 0.24) even though
the model is the same - the test weakens as it is overfitted, and at the
textbook extreme it ends at the p ~ 1.0 red flag. The uncollapsed 2:3
spec is also instructive: Hansen REJECTS it (p < 0.05) while its
collapsed twin passes - proliferation distorts the test in both tails.
Collapsing keeps estimates stable with a fraction of the instruments.

Takeaways:
  1. OLS (0.962) and FE (0.626) bracket the truth from above/below.
  2. Anderson-Hsiao is consistent but imprecise (se 0.48).

  4. System GMM (0.927, se 0.079) is inside the bracket with
     clean diagnostics (AR(2) p=0.99, Hansen p=0.46).
  5. Employment is highly persistent: about 93% of a shock survives into the next year.
