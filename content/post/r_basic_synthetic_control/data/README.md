# Data dictionary - Basic Synthetic Control: The Basque Country Case Study

Companion data for a beginner-friendly R tutorial on the <strong>synthetic control method</strong>, applied to the classic Basque Country case study of Abadie and Gardeazabal (2003). The panel is the <code>basque</code> dataset bundled with the <strong>Synth</strong> R package: annual observations from 1955&ndash;1997 for 18 Spanish regional units. The outcome is real GDP per capita (<code>gdpcap</code>, 1986 thousands of USD); the remaining columns are the 13 pre-treatment predictors matched by the algorithm &mdash; six sectoral production shares, five education levels, an investment-to-GDP ratio, and a 1969 population density. The tutorial frames the analysis as a causal problem (estimand: ATT), builds a &ldquo;synthetic Basque&rdquo; from a weighted recipe of the other regions (85% Catalonia, 15% Madrid), and stress-tests the result with Catalonia and in-space placebos.


Companion data for the post <https://carlos-mendez.org/post/r_basic_synthetic_control/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Predictor columns are sparse by design.</strong> Only <code>gdpcap</code> is observed for all 774 rows. The six sectoral shares (~90 non-missing) and five education levels (~108) are pooled period averages/snapshots, and <code>popdens</code> is a single 1969 cross-section (18 rows). The <code>Synth</code> workflow reads these via period-aggregation arguments, so the gaps are expected, not data errors.
- <strong>School columns are raw counts, not shares.</strong> <code>school_illit</code>, <code>school_prim</code>, <code>school_med</code>, <code>school_high</code>, and <code>school_post_high</code> are stored as raw figures; the post converts them to within-region percentage shares (and collapses <code>school_high</code> + <code>school_post_high</code>) before matching.
- <strong>Region 1 is an aggregate.</strong> Row group <code>regionno == 1</code> (&ldquo;Spain (Espana)&rdquo;) is the national total and is dropped from the analysis; the 17 autonomous communities are regions 2&ndash;18.
- <strong>Column names use underscores.</strong> The source <code>basque</code> object names sectoral and school variables with dots (e.g. <code>sec.agriculture</code>); they are renamed to underscores here (<code>sec_agriculture</code>) so the variables are valid Stata names. Values are unchanged.
- <strong>Real data, single treated unit.</strong> This is real regional accounting data, not simulated. With one treated region and a 16-region donor pool, inference rests on placebos rather than standard errors; the discrete pseudo p-values have limited resolution.
