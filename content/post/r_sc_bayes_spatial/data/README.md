# Data dictionary - Bayesian Spatial Synthetic Control: California's Proposition 99

Companion data for an R tutorial that replicates the California case study of Sakaguchi &amp; Tagawa (2026) on cigarette consumption and Proposition 99. The single file is the balanced state-year panel bundled in the <code>scspill</code> replication package &mdash; the same <strong>real</strong> tobacco panel introduced by Abadie, Diamond &amp; Hainmueller (2010): per-capita cigarette sales (<code>cigsale</code>) and real retail price (<code>retprice</code>) for 39 US states over 1970&ndash;2000. California is the one treated unit (Prop 99 switches on in 1988); the other 38 states form the donor pool. The post fits three nested synthetic-control estimators on this panel &mdash; classical SCM (<code>tidysynth</code>), a Bayesian horseshoe-prior SCM, and a Bayesian spatial SCM with a spatial autoregressive (SAR) layer &mdash; and reads the ATT, the donor weights, and the per-state spillovers off each.


Companion data for the post <https://carlos-mendez.org/post/r_sc_bayes_spatial/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Narrow predictor set.</strong> The shipped panel carries only <code>cigsale</code> and <code>retprice</code> — not the log income, youth-share, or beer-sales predictors Abadie (2010) used. This is the main reason the classical ATT (≈ −18) is smaller in magnitude than Abadie's published ≈ −27.
- <strong>Spatial weights live elsewhere.</strong> The SAR layer needs California's contiguity vector <code>w</code> and the 38×38 donor contiguity matrix <code>W</code>, which ship inside the <code>scspill</code> package, not in this CSV. The data dictionary documents only the outcome panel.
- <strong>Tutorial-scale MCMC.</strong> The post runs 5,000 iterations (vs the paper's 100,000), so the SAR ρ effective sample size is only ~3 and the Stage 3 credible interval is illustrative, not paper-grade. This is an analysis caveat, not a data caveat — the panel itself is exact.
- <strong>Treatment timing.</strong> Prop 99 was approved in November 1988 and took effect January 1989, but the package convention codes 1988 as the first post-treatment year; the <code>treatment</code> dummy follows that convention.
