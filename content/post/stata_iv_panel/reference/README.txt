Replication Material for "Hodler, R & Raschky, P.A. (2014). 'Economic shocks and civil conflict at the regional level'.
Economics Letters, 124(3), pp. 530-533."


- "EL_regional_conflict_replication.do" is the stata .do file to generate the results in Tables 1-4.
- "EL_regional_conflict_replication.dta" is the corresponding stata .dta file.

The data file already contains the detrended variables (*_dt). Lines 22-56 contain the code
to generate the detrended variables.
