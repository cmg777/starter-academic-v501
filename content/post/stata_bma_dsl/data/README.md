# Data dictionary - Taming Model Uncertainty in the Environmental Kuznets Curve

Companion data for a Stata tutorial that confronts the <strong>model-uncertainty problem</strong> in testing the Environmental Kuznets Curve (EKC). With 12 candidate control variables there are <code>2&#8313;&sup2; = 4,096</code> possible regressions on log CO<sub>2</sub> per capita, and choosing one is a hidden assumption. The tutorial applies two principled solutions &mdash; <strong>Bayesian Model Averaging</strong> (BMA, via Stata&rsquo;s <code>bmaregress</code>) and <strong>Post-Double-Selection LASSO</strong> (DSL, via <code>dsregress</code>) &mdash; to estimate the inverted-N cubic relationship between log CO<sub>2</sub> and log GDP. The data is <strong>fully synthetic</strong>, built from a known data-generating process in which <strong>5 controls truly affect emissions and 7 are pure noise</strong>, so each method can be graded against ground truth. With country and year fixed effects, BMA recovers the GDP cubic almost exactly and flags 6 of 8 true predictors at PIP &ge; 0.80 with zero false positives; DSL produces fast cluster-robust estimates. Stripping the fixed effects inflates the GDP coefficient and generates false positives.


Companion data for the post <https://carlos-mendez.org/post/stata_bma_dsl/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Synthetic data.</strong> There is no real data behind this tutorial. Every value is simulated from a known data-generating process; results are internally consistent with the calibration but are NOT empirical evidence about real-world CO<sub>2</sub> emissions or the EKC.
- <strong>Inspired by, not identical to.</strong> The panel is inspired by Gravina & Lanzafame (2025) but the values are fully synthetic and do not reproduce that study's data.
- <strong>Known answer key.</strong> Exactly 5 controls (<code>fossil_fuel, renewable, urban, democracy, industry</code>) have a true non-zero effect; the other 7 are pure noise with a true coefficient of zero. Use this only to grade method accuracy, not to draw substantive conclusions.
- <strong>Weak signals are hard.</strong> Two true controls (<code>urban</code> +0.007, <code>democracy</code> -0.005) have small coefficients and fall below the PIP &ge; 0.80 threshold even though they belong &mdash; a realistic limitation, not a data error.
- <strong>Fixed effects matter.</strong> The cubic-EKC recovery and zero-false-positive selection hold only with country and year fixed effects; pooled (no-FE) estimates inflate the GDP coefficient 2&ndash;3&times; and generate false positives.
- <strong>Country IDs are arbitrary.</strong> <code>country_id</code> 1&ndash;80 are synthetic identifiers with no real-country mapping; the CSV rows are not sorted by id.
