# Data dictionary - Latent Group Structures in Panel Data

Companion data for a hands-on Stata tutorial that uses the <strong>Classifier-LASSO</strong> (C-LASSO) method of Su, Shi &amp; Phillips (2016), via the <code>classifylasso</code> command (Huang, Wang &amp; Zhou 2024), to discover <em>latent group structures</em> in panel data &mdash; subsets of units that share slope coefficients while groups differ. Two source datasets drive the post. <code>saving</code> is a balanced panel of 56 countries observed over 15 years (840 observations) on savings behavior, from the Su&ndash;Shi&ndash;Phillips (2016) replication. <code>democracy</code> is the Acemoglu, Naidu, Restrepo &amp; Robinson (2019) democracy-and-growth panel of 98 countries from 1970&ndash;2010 (4,018 observations). The tutorial shows that the pooled democracy&ndash;growth effect of +1.055 masks a +2.151 effect in 57 countries and a &minus;0.936 effect in 41 countries &mdash; a sign reversal exemplifying Simpson's paradox.


Companion data for the post <https://carlos-mendez.org/post/stata_panel_lasso_cluster/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Standardized savings variables.</strong> In <code>saving.dta</code>, savings, lagsavings, cpi, interest, and gdp are each standardized to mean 0, SD 1 — coefficients are in standard-deviation units, not raw economic magnitudes.
- <strong>Coded time index.</strong> In <code>saving.dta</code>, <code>year</code> is coded 1–15 (not calendar years); these 15 periods correspond to 1995–2010 in the source.
- <strong>Numeric country codes.</strong> Both panels identify units by numeric code (<code>code</code> in savings; <code>country</code> in democracy). The democracy file carries a <code>name</code> string; the savings file does not, so savings groups cannot be mapped to named countries.
- <strong>Scaled GDP variables.</strong> In <code>democracy.dta</code>, <code>lnPGDP</code> and its lags <code>ly1–ly4</code> are log per-capita GDP multiplied by 100; <code>lnTrade</code> and <code>lnMort</code> are likewise scaled by 100.
- <strong>Associations, not proven causation.</strong> The group-specific C-LASSO estimates are conditional associations within the panel model; a causal reading of the democracy effect requires the same identifying assumptions as Acemoglu et al. (2019).
- <strong>Close IC in the democracy model.</strong> IC values across K = 1–5 span only 3.267–3.280; K = 2 is optimal but not dominant, so group assignments are sensitive to the tuning parameter ρ.
