# Data dictionary - Double LASSO for Causal Inference: Does Abortion Reduce Crime?

Companion data for a hands-on R tutorial that replicates the empirical example in Fitzgerald, Lattimore, Robinson &amp; Zhu (2026, <em>Journal of Applied Econometrics</em>), which itself builds on <a href="https://doi.org/10.1093/restud/rdt044" target="_blank" rel="noopener">Belloni, Chernozhukov &amp; Hansen (2014)</a> and the controversial <a href="https://doi.org/10.1162/00335530151144050" target="_blank" rel="noopener">Donohue &amp; Levitt (2001)</a> finding that legalized abortion reduced U.S. crime. The panel is 48 U.S. states over 12 first-differenced years (1986&ndash;1997, 576 observations) after the raw 13-year (1985&ndash;1997) panel is first-differenced to absorb state fixed effects. The treatment is the <strong>effective abortion rate</strong>; the three outcomes are <strong>violent crime, property crime, and murder</strong> rates; and the candidate-control set has <strong>284 columns</strong> per outcome. The post compares five estimators &mdash; first-difference OLS, kitchen-sink OLS, Post-Structural LASSO, and Double LASSO under both the rigorous (<code>hdm::rlasso</code>) and cross-validated (<code>glmnet::cv.glmnet</code>) penalties &mdash; each with HC1 state-clustered standard errors. These CSVs are the inputs the post's <code>analysis.R</code> loads over HTTPS; they were extracted from the paper's Matlab replication archive by the companion <code>prepare_data.R</code>.


Companion data for the post <https://carlos-mendez.org/post/r_double_lasso/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Datasets

| File | Grain | Rows x Cols | Purpose |
|---|---|---|---|
| `levitt_state` | state-year (one row per observation) | 576 x 1 | Cluster id for the state-clustered (HC1) standard errors. |
| `levitt_linear` | state-year (first-differenced) | 576 x 7 | First-difference OLS baseline (no controls) for each outcome. |
| `levitt_partialled` | state-year (partialled) | 576 x 7 | Treatment/outcome inputs to the LASSO regressions (no time dummies). |
| `levitt_controls_viol` | state-year x control | 576 x 284 | The 284 candidate controls the LASSO selects from for violent crime. |
| `levitt_controls_prop` | state-year x control | 576 x 284 | The 284 candidate controls the LASSO selects from for property crime. |
| `levitt_controls_murd` | state-year x control | 576 x 284 | The 284 candidate controls the LASSO selects from for murder. |
| `levitt_control_names` | control index (1-284) | 284 x 4 | Map each control-matrix column position to its variable name per outcome. |

## Known limitations & caveats

- <strong>Replication, not a primary causal claim.</strong> Fitzgerald et al. (2026) is itself a methods replication; whether abortion access caused less crime is a substantive debate that goes well beyond any single regression. The data is borrowed as a clean case study of Double LASSO, not to endorse the Donohue–Levitt 2001 claim.
- <strong>Identification rests on two assumptions.</strong> Conditional independence given the 284 partialled controls, and parallel trends in levels (state FE absorbed by first-differencing, year FE by the partialling step). Neither is innocuous.
- <strong>Non-syntactic column names.</strong> The control matrices keep the original Matlab names (containing <code>^</code> and <code>*</code>), so the post loads them with <code>check.names = FALSE</code>. The labeled Stata <code>.dta</code> in this bundle sanitizes those characters to underscores (<code>Lprison^2</code> → <code>Lprison_2</code>) so the names are valid Stata identifiers; the source CSVs keep the originals.
- <strong>State-clustering relies on G ≥ 30.</strong> Cluster-robust inference is justified asymptotically in the number of clusters; with G = 48 states it is above the rule of thumb.
- <strong>CV LASSO is non-deterministic.</strong> cv.glmnet randomly partitions folds; the post sets set.seed(20260520) so its numbers reproduce. The rigorous LASSO is deterministic given the data and penalty arguments.
- <strong>Equal weighting.</strong> Every state-year observation gets equal weight; the estimand is not population-weighted.
