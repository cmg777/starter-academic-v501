# Data dictionary - MGWFER: Causal Spatially Varying Coefficients via Panel Fixed Effects

Companion data for a Python tutorial faithful to <strong>Li &amp; Fotheringham (2026)</strong>, which introduces <strong>Multiscale Geographically Weighted Fixed Effects Regression (MGWFER)</strong> &mdash; a local panel framework that removes time-invariant spatial confounders from Multiscale GWR. The dataset is a <strong>fully synthetic</strong> spatial panel generated verbatim from the paper&rsquo;s data-generating process (Eqs. 39&ndash;45) on a 15&times;15 grid of 225 spatial units observed over 3 time periods (675 observations). Each covariate is coupled to a time-invariant spatial context <code>sc_i</code> (<code>Cor(x_k, sc) &asymp; 0.84</code>), so the <em>indirect contextual effect</em> channel is active and <code>Cor(x_4, y) &asymp; 0.84</code> even though <code>&beta;_4 &equiv; 0</code>. Because the true coefficient surfaces (<code>beta1_true</code>&ndash;<code>beta4_true</code>) and the confounder (<code>alpha_true</code>) are carried as columns, the panel is ground truth against which OLS, pooled OLS, fixed effects, cross-sectional MGWR, pooled MGWR, and MGWFER can be benchmarked. The entire data-generating process is open and reproducible.


Companion data for the post <https://carlos-mendez.org/post/python_mgwrfer/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Synthetic data.</strong> There is no real data behind this tutorial; values are simulated from the paper's DGP with a fixed seed. Results are internally consistent with the design but are not empirical evidence about real-world spatial processes.
- <strong>Truth columns are not observable in practice.</strong> <code>alpha_true</code> and <code>beta1_true</code>&ndash;<code>beta4_true</code> are carried only so estimators can be scored. In an applied setting these are exactly the unknowns the methods try to recover — do not feed them to the models as predictors.
- <strong>Indirect channel by construction.</strong> Every covariate is 84% correlated with spatial context, and <code>x4</code> is 84% correlated with <code>y</code> despite β₄ = 0. Any model that fails to condition on <code>sc</code> (cross-sectional/pooled OLS, MGWR_cs, PMGWR) will misread this as a real local effect — that is the point of the simulation, not a data error.
- <strong>Reduced 15×15 grid.</strong> The paper uses a 30×30 grid; this panel uses 15×15 (225 units) so the multiscale bandwidth search completes in minutes. All qualitative conclusions are preserved, but exact RMSE/correlation magnitudes differ from the paper's.
- <strong>Balanced, short panel.</strong> Only T = 3 periods; the within estimator's degrees of freedom are NT − K − N = 446. Fixed-effects identification relies on strict exogeneity conditional on the fixed effects (no time-varying confounders).
