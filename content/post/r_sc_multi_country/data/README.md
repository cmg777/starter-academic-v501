# Data dictionary - Augmented Synthetic Control for Multiple Countries

Companion data for a hands-on R tutorial on the <strong>Augmented Synthetic Control Method (ASCM)</strong> of Ben-Michael, Feller &amp; Rothstein (2021), demonstrated in a multi-country setting with the <code>augsynth</code> package and its three entry points (<code>single_augsynth</code>, <code>multisynth</code>, <code>augsynth_multiout</code>). Both files here are <strong>fully synthetic</strong>: the true counterfactual and injected effect are known by construction, so every estimate can be graded against ground truth before the method is turned loose on the real Penn-World-Table EMU data (which lives in the post's <code>reference/</code> folder and is not documented here). The simulated panels validate that each estimator recovers a known effect and show exactly where plain SCM fails and Ridge augmentation rescues it.


Companion data for the post <https://carlos-mendez.org/post/r_sc_multi_country/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Synthetic data.</strong> Both files are fully simulated; results are internally consistent with the calibration but are not empirical evidence about real-world policy effects. The real euro-area data underlying Part 2 is the Penn World Table panel in the post's reference/ folder, which is not documented here.
- <strong>C05 is outside the donor hull by design.</strong> No convex blend of the donors can reproduce its pre-treatment path, so plain SCM mis-signs its effect; this is intentional, to show where Ridge augmentation earns its keep.
- <strong>Ground-truth columns ship with the data.</strong> gdp_index_cf / trade_index_cf (the true counterfactuals) and true_effect_gdp / true_effect_trade (the injected effects) are revealed here for grading; real datasets never give you these.
- <strong>Significance is a choice.</strong> On these simulated panels the injected effects are real, so headlines are significant — but the jackknife and the wild bootstrap can still disagree (the pooled multisynth effect is significant under the jackknife yet not under the bootstrap). Match the inference tool to the estimator and report when they disagree.
