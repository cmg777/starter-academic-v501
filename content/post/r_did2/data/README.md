# Data dictionary - Difference-in-Differences for Regional Data

Companion data for a hands-on R tutorial that asks whether the Affordable Care Act's staggered Medicaid expansion reduced adult mortality, and uses the question to show how <strong>population weighting changes the target parameter</strong> when the units (U.S. counties) differ in size by orders of magnitude. Following Baker, Callaway, Cunningham, Goodman-Bacon and Sant'Anna's (2025) <em>Practitioner's Guide</em>, the post runs an eight-stage DiD pipeline &mdash; 2&times;2 cell means, three equivalent TWFE specifications, covariate-adjusted OR/IPW/DRDID, a 2&times;T event study, the Callaway&ndash;Sant'Anna staggered ATT(g,t) design, and a Rambachan&ndash;Roth HonestDiD sensitivity analysis &mdash; computing every estimate both unweighted and weighted by 2013 adult population. The headline 2&times;2 ATT(2014) flips sign with weighting, from +0.122 deaths per 100,000 unweighted to &minus;2.563 weighted, while no 95% confidence interval at any stage comfortably excludes zero. The two estimands answer different questions &mdash; the typical treated <em>county</em> versus the typical treated <em>adult</em>.


Companion data for the post <https://carlos-mendez.org/post/r_did2/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Pedagogical, not definitive.</strong> The authors of the replicated guide flag this case as illustrative: "The results are pedagogical in spirit and do not represent the best possible estimates of Medicaid's effect on adult mortality."
- <strong>Underpowered.</strong> None of the six 2x2 covariate-adjusted 95% confidence intervals excludes zero; the data cannot settle the policy question.
- <strong>Weighting changes the estimand.</strong> Unweighted (ATT for the typical treated county) and population-weighted (ATT for the typical treated adult) answer different causal questions; the 2x2 ATT(2014) flips sign between them (+0.122 vs -2.563).
- <strong>Crude, not age-adjusted.</strong> The outcome is the CDC crude death rate for ages 20-64, not an age-adjusted rate, so compositional differences across cohorts are not removed.
- <strong>Small cohorts are noisy.</strong> The 2015, 2016, and 2019 expansion cohorts are small (171 / 93 / 140 counties); cohort-specific estimates carry wide confidence intervals.
- <strong>Tutorial bootstrap.</strong> The Callaway-Sant'Anna bootstrap uses 2,000 iterations for speed (the reference scripts use 25,000), affecting the third significant figure of each CI.
