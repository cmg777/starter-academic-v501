# Data dictionary - Causal Inference with DoWhy: The Lalonde Job-Training Data

Companion data for a hands-on Python tutorial that estimates the causal effect of a job-training program on earnings using <strong>DoWhy</strong>'s four-step framework (Model &rarr; Identify &rarr; Estimate &rarr; Refute). The file is the <strong>Lalonde sample</strong> from the U.S. <strong>National Supported Work (NSW) Demonstration</strong>, a 1970s randomized employment program for disadvantaged workers (LaLonde 1986; the experimental subsample popularized by Dehejia &amp; Wahba 1999). It records 445 participants &mdash; 185 randomly assigned to job training (treatment) and 260 to control &mdash; with eight pre-treatment covariates and real earnings in 1978 (<code>re78</code>) as the outcome. The post encodes the eight covariates as common causes in a causal graph, identifies the backdoor estimand, and estimates the average treatment effect with five methods (regression adjustment, IPW, doubly robust AIPW, propensity-score stratification, and propensity-score matching), all clustering near a roughly 34&ndash;38% earnings gain over the control mean. These are <strong>real data</strong> from a field experiment, not a simulation.


Companion data for the post <https://carlos-mendez.org/post/python_dowhy/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Small sample.</strong> Only 445 observations (185 treated / 260 control); propensity-score methods can suffer from poor overlap and estimates carry high variance.
- <strong>Earnings are right-skewed with mass at zero.</strong> <code>re74</code>, <code>re75</code>, and <code>re78</code> have a large spike at $0 (no earnings) and a long right tail, so means exceed medians substantially.
- <strong>Unconfoundedness.</strong> The backdoor estimand assumes the eight covariates capture all confounding; this is credible here because assignment was randomized, but the same data is often used to study what happens when it is not.
- <strong>ATE vs ATT.</strong> Four estimators target the ATE; propensity-score matching discards unmatched controls and shifts the estimand toward the ATT, so its estimate answers a slightly different question.
- <strong>Experimental subsample.</strong> This is the randomized NSW experimental sample (Dehejia-Wahba), not the larger observational PSID/CPS comparison versions; results are not directly comparable to those non-experimental variants.
