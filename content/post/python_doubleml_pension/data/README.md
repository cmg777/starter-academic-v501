# Data dictionary - Double Machine Learning with 401(k) Pension Data

Companion data for a hands-on Python tutorial that estimates the causal effect of 401(k) eligibility and participation on household net financial assets using <strong>Double Machine Learning</strong> (DoubleML). The dataset is the <strong>real</strong> 401(k) sample drawn from the 1991 U.S. Survey of Income and Program Participation (SIPP) &mdash; the same nationally representative cross-section of 9,915 households used by Chernozhukov et&nbsp;al. (2018) and distributed with the DoubleML package via <code>fetch_401K</code> (originally constructed from the SIPP by Poterba, Venti &amp; Wise). About 37&nbsp;% of households are eligible for a 401(k) and 26&nbsp;% participate. The post fits three estimators &mdash; Partially Linear Regression (PLR) and the doubly robust Interactive Regression Model (IRM), both targeting the Average Treatment Effect of eligibility, and the Interactive IV Model (IIVM), which instruments participation with eligibility to recover the Local Average Treatment Effect on compliers &mdash; each fit with four ML learners (Lasso, Random Forest, Decision Tree, XGBoost) under cross-fitting.


Companion data for the post <https://carlos-mendez.org/post/python_doubleml_pension/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Real survey data, single snapshot.</strong> These are real 1991 SIPP households, not simulated; but the cross-section is one point in time, so dynamic effects of 401(k) participation over time are not identified.
- <strong>Conditional exogeneity.</strong> The ATE estimates assume eligibility is as good as randomly assigned after conditioning on the observed covariates. Unobserved factors (e.g. financial literacy) that affect both eligibility and savings would bias the estimates.
- <strong>Extreme asset values.</strong> <code>net_tfa</code> ranges from −$502,302 to $1,536,798 (and <code>tw</code> up to $2,029,910); a handful of outliers influence the mean-based ATE estimates.
- <strong>IIVM identifies the LATE, not the ATE.</strong> The ≈$11,746 IIVM estimate applies only to compliers (households who participate because eligible) and should not be generalized to the full population without the monotonicity assumption.
- <strong>nifa / tw are extras.</strong> The tutorial's outcome is <code>net_tfa</code>; <code>nifa</code> (non-401(k) financial assets) and <code>tw</code> (total wealth) ship in the source file but are not modeled — included here for completeness.
