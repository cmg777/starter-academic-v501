# Plan: Synthetic Control Method Stata Script

## Context

Create a Stata tutorial script (`analysis.do`) for the synthetic control method, using California's Proposition 99 tobacco control program as the case study. Reference materials exist in `content/post/stata_sc/reference/` containing working `synth2` code and pedagogical explanations from a Notion export. The goal is to create a comprehensive script that covers the SCM methodology and its inference approaches (placebo tests, leave-one-out), following the project's established conventions for Stata tutorials.

No `index.md` or `analysis.do` currently exists in `content/post/stata_sc/` -- only the reference materials.

## Approach: Invoke `/project:write-script`

Use the write-script skill with these parameters:
- **Topic:** Synthetic control method -- effect of California's Proposition 99 on cigarette sales
- **Dataset:** `smoking_sc.dta` from `https://github.com/quarcs-lab/data-open/raw/master/isds/smoking_sc.dta`
- **Language:** Stata
- **References:** The Notion export at `content/post/stata_sc/reference/`

## Script Structure (~450-500 lines)

Following conventions from `stata_did/analysis.do` (524 lines, 13 sections):

### Header Block (~45 lines)
- Title, reference (Abadie, Diamond & Hainmueller 2010), URL, dataset description, variables list, estimand (ATT on California), usage, required packages (synth, synth2), Apple Silicon Rosetta note

### Section 0: Install dependencies + start log (~20 lines)
- `capture ssc install synth, all replace`
- `capture ssc install synth2, all replace`
- `capture log close` / `log using "analysis.log", replace text`
- Banner display

### Section 1: Load and explore dataset (~45 lines)
- `use "https://...smoking_sc.dta", clear` + `describe` + `summarize` + `xtset state year` + `xtsum`
- `label list` to identify California as state==3
- Display text explaining panel structure (39 states, 1970-2000) and variables

### Section 2: Visualize raw trends (~50 lines)
- Create line plot: California vs. average of donor pool states over time
- `xline(1989)` for treatment onset
- **Figure:** `stata_sc_raw_trends.png`

### Section 3: Baseline synthetic control estimate (~55 lines)
- Core `synth2` command from reference with `nested allopt`
- `ereturn list` to show stored results
- Display predictor balance table and unit weights
- Interpretive text explaining command arguments
- **Figures:** Auto-generated by synth2: `pred`, `eff`, `bias`, `weight_unit`, `weight_vars` -- export each as `stata_sc_*.png`

### Section 4: Interpret predictor balance and weights (~40 lines)
- `matrix list e(X)` (predictor balance), `matrix list e(W)` (unit weights)
- Display interpretive text: which states receive highest weights, quality of predictor match

### Section 5: Interpret treatment effects (~30 lines)
- Display estimated treatment effect magnitude
- Interpretive text about pre-treatment fit and post-treatment divergence

### Section 6: In-space placebo test (~50 lines)
- `synth2 ... placebo(unit cut(2)) sigf(6)` from reference
- Explain: each control state treated as if it were the treated unit; MSPE filter
- Display p-values and interpretation
- **Figures:** `eff_pboUnit`, `ratio_pboUnit`, `pvalTwo_pboUnit`, `pvalRight_pboUnit`, `pvalLeft_pboUnit` -- export each

### Section 7: In-time placebo test (~40 lines)
- `synth2 ... placebo(period(1985))` from reference
- Explain: fake treatment in 1985, drop cigsale(1988) predictor, xperiod ends at 1984
- Interpret: should see NO effect at fake treatment date
- **Figures:** `pred_pboTime1985`, `eff_pboTime1985` -- export each

### Section 8: Leave-one-out robustness check (~45 lines)
- `synth2 ... loo frame(california) savegraph(california, replace)` from reference
- `graph combine` for combined LOO display
- Explain: sensitivity to individual donor states
- **Figures:** LOO combined graph

### Section 9: Inspect leave-one-out frame (~25 lines)
- `frame change california`, `describe`, `frame change default`
- Explain frame contents

### Section 10: Closing summary (~35 lines)
- Key findings with specific numbers
- Inference summary
- List of all exported figures
- Reference citation
- `log close`

## All Reference Code Blocks (must be included)

1. Package install: `ssc install synth` / `ssc install synth2`
2. Data: `use "https://...smoking_sc.dta", clear` + `describe` + `xtset state year`
3. Baseline: `synth2 cigsale lnincome age15to24 retprice beer cigsale(1988) cigsale(1980) cigsale(1975), trunit(3) trperiod(1989) xperiod(1980(1)1988) nested allopt`
4. In-space placebo: `synth2 ... placebo(unit cut(2)) sigf(6)`
5. In-time placebo: `synth2 cigsale lnincome age15to24 retprice beer cigsale(1980) cigsale(1975), trunit(3) trperiod(1989) xperiod(1980(1)1984) nested placebo(period(1985))`
6. LOO: `synth2 ... loo frame(california) savegraph(california, replace)`
7. Graph combine: `` graph combine `e(graph)', cols(2) altshrink ``
8. Frame inspect: `frame change california` / `describe` / `frame change default`
9. Apple Silicon check: `display c(machine_type)`
10. Label list: `label list`
