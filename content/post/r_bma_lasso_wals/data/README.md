# Data dictionary - Robust Variable Selection: BMA, LASSO, and WALS

Companion data for a hands-on R tutorial comparing three principled responses to the variable selection problem &mdash; <strong>Bayesian Model Averaging (BMA)</strong>, the <strong>LASSO</strong>, and <strong>Weighted Average Least Squares (WALS)</strong> &mdash; on a <strong>fully synthetic</strong> cross-section of 120 fictional countries. Twelve candidate regressors compete to explain log CO<sub>2</sub> emissions: <strong>7 have true nonzero effects</strong> and <strong>5 are pure noise</strong> deliberately correlated with GDP and the true predictors, creating realistic multicollinearity. Because the data-generating process is known, the data carries its own &ldquo;answer key&rdquo; against which each method is graded. The convergence of mechanically distinct methods on the same variables &mdash; four are flagged by all three (triple-robust) &mdash; illustrates methodological triangulation. The entire DGP is open and reproducible (<code>set.seed(2017)</code>).


Companion data for the post <https://carlos-mendez.org/post/r_bma_lasso_wals/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Synthetic data.</strong> There is no real data behind this tutorial. The 120 &ldquo;countries&rdquo; and all 13 numeric variables are simulated; results are internally consistent with the calibrated DGP but are not empirical evidence about real CO<sub>2</sub> emissions or their determinants.
- <strong>Built-in answer key.</strong> The true coefficients are known by design (7 nonzero, 5 exactly zero). The point of the dataset is to grade each method against this truth; that luxury does not exist with real data, which is exactly why the post recommends using all three methods.
- <strong>Deliberate multicollinearity.</strong> The noise variables are constructed to be correlated with GDP and the true predictors, so a naive OLS can show spurious significance. This is intentional, to make variable selection genuinely hard.
- <strong>Power, not bias, limits detection.</strong> All methods reach perfect specificity, but small true effects (agriculture, β = 0.005; and the borderline urban_pop / democracy for BMA) are hard to detect at n = 120 — a sample-size limitation, not a flaw in any method.
- <strong>Cross-section only.</strong> No time dimension and no panel structure; the methods here address model uncertainty, not endogeneity, nonlinearity, or heteroskedasticity, which real applications must handle separately.
