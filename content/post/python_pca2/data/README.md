# Data dictionary - Pooled PCA for Building Development Indicators Across Time

Companion data for a hands-on Python tutorial that builds a <strong>temporally comparable</strong> composite Human Development Index using <strong>pooled PCA</strong> and contrasts it with the naive per-period approach. The data are a South American subset of the <a href="https://globaldatalab.org/shdi/">Subnational Human Development Index</a> from the Global Data Lab (<a href="https://doi.org/10.1038/sdata.2019.38">Smits &amp; Permanyer, 2019</a>), with Education, Health, and Income sub-indices (plus the official SHDI and its underlying components) for 153 sub-national regions across 12 South American countries in 2013 and 2019. Pooled PCA stacks both periods and computes a single set of standardization parameters, eigenvector weights, and normalization bounds from the combined data, so a region's 2013 score is measured against the same yardstick as its 2019 score. The pooled PC1 captures 72.42&#37; of variance with weights [0.5642, 0.5448, 0.6204] and registers a net development shift of +0.1439 that per-period PCA forces to zero by construction.


Companion data for the post <https://carlos-mendez.org/post/python_pca2/>. Each dataset ships as a labeled Stata `.dta` (v118) plus its source file. Generated by `build_data_dictionary.py`.

## Known limitations & caveats

- <strong>Real data.</strong> Values are the Global Data Lab's published Subnational HDI for South America (Smits &amp; Permanyer, 2019); they are not simulated.
- <strong>Two periods only.</strong> 2013 and 2019 are the minimum for temporal analysis; more periods would strengthen the pooled estimates and allow testing the constant-correlation assumption.
- <strong>Sample-relative index.</strong> The PCA-based HDI is relative to this specific 153-region sample &mdash; adding or removing regions changes every score, and Min-Max normalization is sensitive to outliers (Potaro-Siparuni, Guyana anchors the bottom).
- <strong>South America only.</strong> Correlation structure and eigenvector weights are specific to this region; other world regions can produce different weights.
- <strong>Income decline is genuine.</strong> Average income fell 2013&rarr;2019 (driven by Venezuela's collapse, 0.782&rarr;0.630), which is why a fixed (pooled) yardstick is needed to compare scores across time.
