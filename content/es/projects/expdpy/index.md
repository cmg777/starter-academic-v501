## expdpy: explorar datos de panel de forma interactiva

**expdpy** es un conjunto de herramientas de análisis exploratorio de datos para datos de panel y de corte transversal. Combina funciones analíticas componibles —que devuelven figuras interactivas de Plotly y tablas de calidad de publicación generadas con Great Tables— con dos aplicaciones web sin código construidas con Streamlit y Shiny para Python. Es una adaptación a Python del paquete de R ExPanDaR.

### Primeros pasos

- **[Cuaderno de inicio rápido (Colab)](https://colab.research.google.com/github/cmg777/expdpy/blob/main/notebooks/quickstart.ipynb)** — ejecute el paquete de principio a fin en su navegador, sin necesidad de instalación.
- **[Aplicación en línea (Streamlit)](https://expdpy.streamlit.app/)** — explore los conjuntos de datos incluidos de Kuznets y Gapminder sin escribir código.
- **[Referencia de la API](https://cmg777.github.io/expdpy/reference/)** — documentación completa a nivel de funciones.

pip install "git+https://github.com/cmg777/expdpy.git"
