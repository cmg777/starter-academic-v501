## expdpy — パネルデータをインタラクティブに探索

**expdpy** は、パネルデータおよびクロスセクションデータのための探索的データ分析ツールキットです。組み合わせ可能な分析関数（インタラクティブなPlotlyの図と、Great Tables による出版品質の表を返します）と、StreamlitおよびShiny for Python で構築された2つのノーコードのウェブアプリを兼ね備えています。R パッケージ ExPanDaR の Python 移植版です。

### はじめに

- **[クイックスタート用ノートブック（Colab）](https://colab.research.google.com/github/cmg777/expdpy/blob/main/notebooks/quickstart.ipynb)** — インストール不要で、ブラウザ上でパッケージを最初から最後まで実行できます。
- **[オンラインアプリ（Streamlit）](https://expdpy.streamlit.app/)** — 付属の Kuznets および Gapminder データセットを、コードを書かずに探索できます。
- **[API リファレンス](https://cmg777.github.io/expdpy/reference/)** — 関数レベルの完全なドキュメント。

### インストール

```bash
pip install "git+https://github.com/cmg777/expdpy.git"
```
