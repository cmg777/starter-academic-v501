## Welcome to metricsAI!

This data science platform offers a modern introduction to econometrics by combining [cloud-based Python notebooks](https://colab.research.google.com/notebooks/empty.ipynb) with [AI learning tools from NotebookLM](https://notebooklm.google.com/notebook/25c819f8-8afc-49aa-8707-2bd87c18d760).

Designed as an interactive companion to Colin Cameron's textbook, [*Analysis of Economics Data: An Introduction to Econometrics*](https://cameron.econ.ucdavis.edu/aed/index.html), metricsAI transforms static chapters into dynamic learning experiences. Students can access AI summaries, Python code, and practical examples directly in their browsers via Google Colab. There is **zero setup or installation required**, and the notebooks feature built-in AI tools for code generation, explanation, and transformation.
