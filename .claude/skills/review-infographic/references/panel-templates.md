# Panel Templates: Narrative Arc Structure

> This file is part of the `review-infographic` skill. Read this file when
> verifying panel structure, storyboard format, and template alignment.

## Template A -- Causal Inference

Use when post tags include "causal", or methods like DML/DoWhy/IV/RCT/partial
identification are central.

| Panel | Dramatic function | Content label | What to verify |
|-------|------------------|---------------|----------------|
| 1 | **Hook** | The Problem | Poses the question or puzzle |
| 2 | **Stakes** | Case Study | Introduces data, sample, what's at risk |
| 3 | **First attempt** | Core Method | Shows how the method works |
| 4 | **Twist** | Comparisons | Compares methods (Comparison metaphor required) |
| 5 | **Surprise** | Key Insight | Most counterintuitive finding |
| 6 | **Resolution** | Bottom Line | Practical guidance, answers guiding question |

## Template B -- ML / Prediction

Use when post focuses on prediction, classification, or model performance.

| Panel | Dramatic function | Content label | What to verify |
|-------|------------------|---------------|----------------|
| 1 | **Hook** | The Question | Prediction target, why it matters |
| 2 | **Stakes** | The Data | Dataset, features, scope |
| 3 | **First attempt** | The Model | Architecture, baseline performance |
| 4 | **Twist** | Tuning / Validation | What improved (Comparison metaphor required) |
| 5 | **Surprise** | Feature Insights | What the model reveals |
| 6 | **Resolution** | Bottom Line | Performance limits, when to use |

## Template C -- Exploratory / Descriptive

Use when post focuses on data patterns, spatial analysis, or descriptive
statistics.

| Panel | Dramatic function | Content label | What to verify |
|-------|------------------|---------------|----------------|
| 1 | **Hook** | The Question | Pattern investigated, why it matters |
| 2 | **Stakes** | The Data | Dataset, scope, key variables |
| 3 | **First pattern** | Pattern A | First finding with key number |
| 4 | **Second pattern** | Pattern B | Contrasting finding (Comparison metaphor required) |
| 5 | **Synthesis** | Connection | How patterns relate |
| 6 | **Resolution** | Implications | Policy implications, next steps |

## What to check during review

**Storyboard format (new):**
- Each panel has exactly: title (3-5 words), central sketch (metaphor), callout (under 8 words), connector
- Central sketches are metaphorical illustrations (NOT precise statistical charts)
- No body text or mini-viz with data values in Section A
- Exactly 3 of 6 callouts contain a BIG number
- No two panels use the same metaphor type
- Panel descriptions are 40-60 words each

**Narrative arc:**
- Story Spine present in Section D
- Dramatic functions assigned (Hook -> Stakes -> Attempt -> Twist -> Surprise -> Resolution)
- Story beats form a coherent beginning-middle-end arc
- Panel 6 answers the guiding question from the title banner

**Template alignment:**
- Correct template selected for the post's topic
- Panel purposes match the template specification
- Panel 4 uses a Comparison metaphor with teal highlight
